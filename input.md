# Instructions

-   **Try to use OOP Principles.**

-   **Naming conventions should be followed in every case; either
    class/component name, variable name, method/function name, parameter
    name or any HTML class or id.**

    -   **DON'T' USE ANY THING LIKE abc AS VARIABLE NAME.**

    -   **Name(Either class, method/function, variable) should depicts
        its purpose**

-   **Code should have Comments**

-   **Starting from 1st assignment use the GitHub repository.**

-   **Demo day: End of every week.**

Note**: Make sure the assignment is complete and everything is working
properly before the demo, if instructed to make some changes in the
previous demo make sure those changes must be applied on the next
demo.**
